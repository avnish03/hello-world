PayPal credentials, make sure payment done on app using PayPal go to your PayPal account. For enabling PayPal you need to enter client id for sandbox and live.
  • Sandbox id is used for testing purposes using paypal sandbox accounts.
  • Live id will be use for paypal live account payments, this is account app user will user.
  • For creating sandbox/live client id, visit https://developer.paypal.com/developer/applications/
