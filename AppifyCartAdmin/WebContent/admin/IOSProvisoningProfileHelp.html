For running an iPhone/iPad, it must be signed with provisioning profile associated with app Bundle id(used in previous step)
  • Create provisioning profile on https://developer.apple.com/
  • There are two types of provisioning profile, development and distribution
  • Development provisioning profile is for testing the app
  • Distribution(or store) provisioning profile is for releasing app store
  • For testing app before releasing app on iTunes, device UDID must be included in provisioning profile
    • For more details check Apple provisioning profile page
